# Basic ToDo with MUI

### Getting Started with Create React App

npx create-react-app dashbord --template typescript

### About
It's a basic todo with React, TS, MUI as like a test task.

User can:
  * Create a new task
  * Edit existing task
  * Delete existing task
  * Makr task as done
  
## What's planned

  * Implement drag-n-drop for increase UX
